# Software3DViewer

Приложение для визуализации, изменения и управления 3D-моделями в формате OBJ с поддержкой камер, текстур и базового освещения.

## Особенности

- **Загрузка 3D-моделей** в формате OBJ
- **Добавление текстур** и изменение базового цвета модели
- **Управление камерами**: несколько камер, орбита, зум, свободное вращение
- **Изменение модели**: позиция, вращение, масштаб
- **Удаление вершин и полигонов** по индексам
- **Режимы рендера**: каркас, текстура, освещение
- **Адаптивный интерфейс**: корректное масштабирование при изменении размера окна

## Требования

- Java 17+
- JavaFX 17+

## Запуск

1. Склонировать репозиторий:
   ```bash
   git clone https://github.com/observernear/Software3DViewer.git
   ```
2. Открыть проект в вашей IDE (IntelliJ IDEA, Eclipse)
3. Настроить SDK на Java 17 и подключить JavaFX
4. Запустить Main.java

## Управление

- **Мышь**:
    - Перетаскивание: вращение камеры вокруг модели (орбита)
    - Колёсико: зум к/от модели

- **Клавиатура**:
    - WASD / стрелки: перемещение камеры по плоскости
    - Shift / Space: перемещение камеры вверх/вниз

- **Панель инструментов**:
    - **Изменение модели**: позиция, вращение, масштаб
    - **Удаление элементов**: выбор вершин или полигонов по индексам
    - **Модели**: управление списком моделей (удаление, выбор активной модели)

- **Режимы рендера**:
    - Полигональная сетка
    - Использование текстуры
    - Освещение
    - Настройка базового цвета модели через ColorPicker
